# Jose Miguel Gómez Lozano

Machine Learning Engineer focused on large language models (LLMs) and applied NLP. I build
AI-based product features end-to-end — from prototype to production, including the
infrastructure that supports it.

I work hands-on with the modern LLM stack: PyTorch, Hugging Face / Transformers, fine-tuning,
embeddings and retrieval-augmented generation (RAG), plus deploying and quantizing models for
efficient, private local inference (Ollama, LM Studio) — and more recently on low-power NPUs
(Intel Core Ultra, OpenVINO, sLLMs).

- **Co-Director @ Sugo Labs** — a family web-development business; I manage our own
